# Mineradio

基于网页端实现的3D立体音乐点歌台，支持网易云音乐、QQ音乐音源。

## 功能列表

- 3D可视化歌单架子，沉浸式点歌体验
- 右键唤起悬浮队列与歌单面板
- 网易云账号登录、搜索、在线歌单与播客
- QQ音乐音源补充与账号登录
- 【Fork新增】本地曲库：自定义文件夹扫描、ID3信息解析、音频流式播放、封面&LRC歌词加载
- 版本自动更新检测
